# 📊 Sales Analytics Dashboard

## 🚀 Project Overview
This is an end-to-end data analytics project where I analyzed sales data using Python and built an interactive dashboard using Power BI.

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib)
- Power BI
- Jupyter Notebook

## 📁 Dataset
The dataset contains sales-related information such as category, region, and monthly sales.

## 🔍 Key Steps
- Data cleaning and preprocessing using Python
- Handling missing values and formatting columns
- Exploratory Data Analysis (EDA)
- Building interactive dashboard in Power BI

## 📊 Dashboard Features
- Total Sales KPI
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Sub-category analysis
- Interactive filters (Category, Region, Month)

## 💡 Insights
- Identified top-performing categories
- Analyzed regional sales distribution
- Observed monthly sales trends
- Highlighted key sub-category performance

## 📌 Conclusion
This project demonstrates my ability to perform data analysis and create interactive dashboards to support business decision-making.
